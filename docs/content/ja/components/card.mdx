Card コンポーネントはカード形式のコンテンツ表示を作成するために使用され、ナビゲーションや機能紹介などのシーンでよく使われます。`Cards` コンポーネントと組み合わせることで、簡単にカードグリッドレイアウトを作成できます。

## 基本的な使い方

`Card` を使用して単一のカードを作成します：

```mdx
<Card title="クイックスタート" href="/quickstart">
  数分でドキュメントサイトを構築できます。
</Card>
```

<Card title="クイックスタート" href="/quickstart">
  数分でドキュメントサイトを構築できます。
</Card>

## カードグリッド

`Cards` で複数の `Card` をラップすると、自動的にグリッドレイアウトになります：

```mdx
<Cards>
  <Card title="設定リファレンス" href="/guide/configuration">
    利用可能なすべての設定オプションを確認する。
  </Card>
  <Card title="ドキュメント作成" href="/guide/writing-docs">
    MDX を使用したドキュメントの書き方を学ぶ。
  </Card>
  <Card title="テーマカスタマイズ" href="/guide/theme">
    ドキュメントサイトの外観をカスタマイズする。
  </Card>
</Cards>
```

<Cards>
  <Card title="設定リファレンス" href="/guide/configuration">
    利用可能なすべての設定オプションを確認する。
  </Card>
  <Card title="ドキュメント作成" href="/guide/writing-docs">
    MDX を使用したドキュメントの書き方を学ぶ。
  </Card>
  <Card title="テーマカスタマイズ" href="/guide/theme">
    ドキュメントサイトの外観をカスタマイズする。
  </Card>
</Cards>

## プロパティ

### Cards

`Cards` はコンテナコンポーネントで、追加のプロパティはありません。複数の `Card` をラップして自動的にグリッドレイアウトに配置するために使用されます。

### Card

| プロパティ | 型 | 必須 | 説明 |
|------|------|------|------|
| `title` | string | はい | カードのタイトル |
| `href` | string | いいえ | クリック時のジャンプ先リンク |
| `children` | ReactNode | いいえ | カードの説明内容 |

## 注意事項

- `Card` と `Cards` は `fumadocs-ui/mdx` のデフォルトコンポーネントであり、追加の import は不要です
- `Card` に `href` が設定されている場合、カードをクリックすると対応するページにジャンプします
- `href` が設定されていないカードはクリックできず、純粋な表示用途に適しています
