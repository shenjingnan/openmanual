# 検索設定

OpenManual には全文検索機能が組み込まれており、デフォルトで有効になっています。

## 検索の有効化／無効化

```json
{
  "search": {
    "enabled": true
  }
}
```

| フィールド | 型 | デフォルト値 | 説明 |
|------|------|--------|------|
| `search.enabled` | boolean | `true` | 検索を有効にするかどうか |

## 検索動作

検索機能は `sidebar` に登録されたすべてのページコンテンツを自動的にインデックス化します。以下が含まれます：

- ページタイトル
- ページ本文の内容
- コードブロック内のテキスト

ユーザーは検索ボックスから関連するコンテンツに素早くアクセスできます。

## 検索結果の最適化

検索結果を改善するためのヒントをいくつか紹介します：

- 各ページに明確な `title` を設定する
- 説明的な見出し階層（h1-h6）を使用する
- 重要な概念の周囲に適切なコンテキストテキストを配置する
- コードブロックに大量の無関係な内容を記載しない
