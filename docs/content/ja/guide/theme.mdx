# テーマのカスタマイズ

OpenManual はシンプルなテーマカスタマイズ機能を提供しており、`theme` 設定フィールドでサイトの外観を調整できます。

## メインカラー

`primaryHue` 色相値を使用してブランドカラーを調整します。色相値の範囲は 0-360 です：

```json
{
  "theme": {
    "primaryHue": 250
  }
}
```

### よく使われる色相値の参考

| 色相値 | 色 | 効果 |
|--------|------|------|
| 0 | 赤 | 情熱的、目立つ |
| 30 | オレンジ | 活力、温かみ |
| 60 | 黄色 | 明るく、親しみやすい |
| 120 | 緑 | 自然、清涼感 |
| 200 | 青 | 専門的、信頼性 |
| 250 | 紫 | 創造的、ミステリアス |
| 300 | ピンク | 優しく、親しみやすい |

## ダークモード

デフォルトでダークモードが有効になっています。設定で無効にすることができます：

```json
{
  "theme": {
    "darkMode": false
  }
}
```

## 完全なテーマ設定

```json
{
  "theme": {
    "primaryHue": 250,
    "darkMode": true
  }
}
```

## テーマ設定フィールド

| フィールド | 型 | デフォルト値 | 説明 |
|------|------|--------|------|
| `primaryHue` | number | — | メインカラーの色相値（0-360） |
| `darkMode` | boolean | `true` | ダークモードを有効にするかどうか |

## 静的リソース

カスタム静的リソース（ロゴ、Favicon など）を `public` ディレクトリに配置します：

```
my-docs/
├── openmanual.json
├── content/
└── public/
    ├── logo.svg
    └── favicon.ico
```

設定内で参照：

```json
{
  "navbar": {
    "logo": "/logo.svg"
  }
}
```

## ロゴのテーマ対応

Light モードと Dark モードで異なるロゴを使用する場合は、オブジェクト形式で設定できます：

```json
{
  "navbar": {
    "logo": {
      "light": "/logo-light.svg",
      "dark": "/logo-dark.svg"
    }
  }
}
```

`light` と `dark` のパスが異なる場合、ナビゲーションバーは 2 つの `<img>` タグを描画し、Tailwind の `dark:hidden` / `hidden dark:block` を利用して自動的に表示を切り替えます。

カスタムロゴ画像が提供されていない場合、OpenManual はサイト名に基づいて自動的に SVG ロゴを生成します。ダークモードでは暖色系の文字色（`#E8E0D4`）が使用され、ダークテーマの背景と調和します。
