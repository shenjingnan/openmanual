# Search Configuration

OpenManual includes built-in full-text search, enabled by default.

## Search Position Configuration

Search is enabled by default, no configuration required. To customize the search entry position:

```json
{
  "search": {
    "position": "header"
  }
}
```

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `search.position` | string | `'sidebar'` | Search entry position: `'sidebar'` (top of sidebar) or `'header'` (center of navbar) |

## Search Behavior

The search feature automatically indexes all pages registered in `sidebar`, including:

- Page titles
- Page body content
- Text within code blocks

Users can quickly locate relevant content through the search box.

## Optimizing Search Results

Here are some suggestions for optimizing search effectiveness:

- Set a clear `title` for each page
- Use descriptive heading hierarchy (h1-h6)
- Use appropriate contextual text around important concepts
- Avoid placing large amounts of irrelevant content in code blocks
