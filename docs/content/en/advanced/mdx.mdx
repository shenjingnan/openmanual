# MDX Enhancements

MDX is a superset of Markdown that allows you to use JSX/React components in your documents. OpenManual provides additional enhancements on top of this.

## LaTeX Formulas

Enable LaTeX support in configuration:

```json
{
  "mdx": {
    "latex": true
  }
}
```

Once enabled, you can use LaTeX syntax in your documents:

### Inline Formulas

Use `$...$` syntax:

```markdown
The mass-energy equation $E = mc^2$ is a famous formula in physics.
```

### Block Formulas

Use `$$...$$` syntax:

```markdown
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

## MDX Configuration Fields

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `mdx.latex` | boolean | `false` | Whether LaTeX formula support is enabled |

## Standard Markdown Support

OpenManual supports all standard Markdown syntax:

- Headings (h1-h6)
- Lists (ordered, unordered)
- Code blocks (with syntax highlighting)
- Tables
- Links and images
- Blockquotes
- Horizontal rules
- Bold, italic, strikethrough

## Code Blocks

Supports syntax highlighting and line markers:

````markdown
```typescript showLineNumbers
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```
````
