# Changelog

All notable project changes are recorded in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/),
versioning follows [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-04-02

### Added

- Initial release
- Zero-config start: only `name` field and `content/index.mdx` needed
- Code generation via template engine producing Next.js apps
- Zod Schema configuration validation with clear error messages
- Multi-group sidebar with custom icons and collapse control
- Theme customization via `primaryHue` brand color adjustment
- Built-in full-text search
- MDX enhancements with LaTeX formula support
- CLI tools (dev / build / preview)
- AI-native design with pure JSON config + Markdown content
