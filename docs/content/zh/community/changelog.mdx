# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2026-04-02

### 新增

- 初始发布
- 零配置起步，最小只需 `name` 字段和 `content/index.mdx`
- 代码生成模式，通过模板引擎生成 Next.js 应用
- Zod Schema 配置校验，错误提示清晰
- 多分组侧边栏，支持自定义图标和折叠控制
- 主题定制，通过 `primaryHue` 调整品牌色
- 内置全文搜索
- MDX 增强，支持 LaTeX 公式
- CLI 工具（dev / build / preview）
- AI 原生设计，纯 JSON 配置 + Markdown 内容
