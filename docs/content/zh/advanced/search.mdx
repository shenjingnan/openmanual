# 搜索配置

OpenManual 内置全文搜索功能，默认开启。

## 启用/禁用搜索

```json
{
  "search": {
    "enabled": true
  }
}
```

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `search.enabled` | boolean | `true` | 是否启用搜索 |

## 搜索行为

搜索功能会自动索引所有在 `sidebar` 中注册的页面内容，包括：

- 页面标题
- 页面正文内容
- 代码块中的文本

用户可以通过搜索框快速定位到相关内容。

## 优化搜索效果

以下是一些优化搜索效果的建议：

- 为每个页面设置清晰的 `title`
- 使用描述性的标题层级（h1-h6）
- 在重要概念周围使用适当的上下文文本
- 避免在代码块中放置大量无关内容
