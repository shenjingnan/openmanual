Accordion 컴포넌트는 접기/펼치기 가능한 콘텐츠 패널을 만드는 데 사용되며, 한정된 공간에서 FAQ, 상세 설명 등 대량의 정보를 표시하는 데 적합합니다.

## FAQ 예제

`Accordions`로 여러 `Accordion`을 감쌉니다:

```mdx
<Accordions>
  <Accordion title="OpenManual이란?">
    OpenManual은 AI 친화적인 오픈소스 문서 시스템 프레임워크입니다.
  </Accordion>
  <Accordion title="설치 방법?">
    pnpm add openmanual으로 설치합니다.
  </Accordion>
</Accordions>
```

<Accordions>
  <Accordion title="OpenManual이란?">
    OpenManual은 AI 친화적인 오픈소스 문서 시스템 프레임워크로, 개발자가 빠르게 아름답고 사용하기 쉬운 문서 사이트를 구축할 수 있도록 돕습니다. MDX를 기반으로 하며, 풍부한 컴포넌트와 커스터마이징 가능한 테마를 지원합니다.
  </Accordion>
  <Accordion title="설치 방법?">
    패키지 관리자로 설치합니다:

    ```bash
    pnpm add openmanual
    ```

    그런 다음 프로젝트 루트 디렉토리에 `openmanual.json` 구성 파일을 생성하면 됩니다.
  </Accordion>
  <Accordion title="어떤 배포 방식을 지원하나요?">
    OpenManual은 정적 사이트를 구축하므로 모든 정적 호스팅 플랫폼에 배포할 수 있습니다:

    - Vercel
    - Netlify
    - GitHub Pages
    - 자체 서버
  </Accordion>
</Accordions>

## 속성

### Accordions

`Accordions`는 컨테이너 컴포넌트로, 추가 속성이 없습니다. 여러 `Accordion` 하위 컴포넌트를 감싸는 데 사용됩니다.

### Accordion

| 속성 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `title` | string | 예 | 패널 제목, 항상 표시됨 |
| `children` | ReactNode | 예 | 패널 콘텐츠, 제목 클릭 시 접기/펼치기 가능 |

## 주의사항

- 각 패널은 기본적으로 접혀 있으며, 제목을 클릭하면 펼쳐집니다
- 패널 콘텐츠는 완전한 Markdown/MDX 구문을 지원합니다
- FAQ, 상세 설명 등 필요에 따라 표시해야 하는 콘텐츠에 적합합니다
- 동일 `Accordions` 내의 여러 패널은 동시에 펼쳐질 수 있습니다
