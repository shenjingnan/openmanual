## [0.1.0] - 2026-04-02

### 추가

- 초기 릴리스
- 영 설정 시작, 최소 `name` 필드와 `content/index.mdx`만 필요
- 코드 생성 모드, 템플릿 엔진을 통한 Next.js 애플리케이션 생성
- Zod Schema 설정 검증, 명확한 오류 메시지 제공
- 다중 그룹 사이드바, 사용자 정의 아이콘 및 접기 제어 지원
- 테마 커스터마이징, `primaryHue`를 통한 브랜드 색상 조정
- 내장 전문 검색
- MDX 확장, LaTeX 수식 지원
- CLI 도구 (dev / build / preview)
- AI 네이티브 디자인, 순수 JSON 설정 + Markdown 콘텐츠
