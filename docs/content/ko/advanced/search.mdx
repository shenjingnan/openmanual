# 검색 설정

OpenManual은 전문 검색 기능을 내장하고 있으며, 기본적으로 활성화되어 있습니다.

## 검색 활성화/비활성화

```json
{
  "search": {
    "enabled": true
  }
}
```

| 필드 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `search.enabled` | boolean | `true` | 검색 활성화 여부 |

## 검색 동작

검색 기능은 `sidebar`에 등록된 모든 페이지 콘텐츠를 자동으로 인덱싱합니다. 다음을 포함합니다:

- 페이지 제목
- 페이지 본문 내용
- 코드 블록의 텍스트

사용자는 검색창을 통해 관련 콘텐츠로 빠르게 이동할 수 있습니다.

## 검색 결과 최적화

검색 품질을 높이기 위한 몇 가지 제안 사항입니다:

- 각 페이지에 명확한 `title` 설정
- 설명적인 제목 계층(h1-h6) 사용
- 중요한 개념 주위에 적절한 맥락 텍스트 사용
- 코드 블록에 대량의 무관한 내용 배치 피하기
