# 사이드바 커스터마이징

사이드바는 문서 사이트의 핵심 네비게이션 컴포넌트로, `openmanual.json`의 `sidebar` 필드를 통해 구성합니다.

## 기본 구조

사이드바는 여러 그룹(group)으로 구성되며, 각 그룹은 여러 페이지(page)를 포함합니다:

```json
{
  "sidebar": [
    {
      "group": "시작하기",
      "pages": [
        { "slug": "index", "title": "프로젝트 소개" },
        { "slug": "quickstart", "title": "빠르게 시작하기" }
      ]
    },
    {
      "group": "가이드",
      "pages": [
        { "slug": "guide/configuration", "title": "설정 참조" }
      ]
    }
  ]
}
```

## 그룹 설정

각 그룹은 다음 필드를 지원합니다:

| 필드 | 타입 | 필수 | 기본값 | 설명 |
|------|------|------|--------|------|
| `group` | string | 예 | — | 그룹 이름 |
| `icon` | string | 아니오 | — | 그룹 아이콘 |
| `collapsed` | boolean | 아니오 | `false` | 기본 접힘 여부 |
| `pages` | array | 예 | — | 그룹 내 페이지 목록 |

## 아이콘

그룹 또는 페이지에 아이콘을 설정하여 네비게이션을 더 직관적으로 만들 수 있습니다:

```json
{
  "sidebar": [
    {
      "group": "시작하기",
      "icon": "Rocket",
      "pages": [
        { "slug": "index", "title": "프로젝트 소개", "icon": "Home" },
        { "slug": "quickstart", "title": "빠르게 시작하기", "icon": "Zap" }
      ]
    }
  ]
}
```

## 접힘 제어

`collapsed: true`를 설정하면 그룹이 기본적으로 접혀 있으며, 사용자 클릭 시 펼쳐집니다:

```json
{
  "sidebar": [
    {
      "group": "고급",
      "collapsed": true,
      "pages": [
        { "slug": "advanced/search", "title": "검색 설정" },
        { "slug": "advanced/mdx", "title": "MDX 고급 기능" }
      ]
    }
  ]
}
```

## 다단계 네비게이션

서브디렉토리를 사용하여 문서를 구성하면 다단계 네비게이션을 구현할 수 있습니다:

```
content/
├── index.mdx
└── guide/
    ├── configuration.mdx
    ├── writing-docs.mdx
    └── theme.mdx
```

대응하는 설정:

```json
{
  "sidebar": [
    {
      "group": "가이드",
      "pages": [
        { "slug": "guide/configuration", "title": "설정 참조" },
        { "slug": "guide/writing-docs", "title": "문서 작성" },
        { "slug": "guide/theme", "title": "테마 커스터마이징" }
      ]
    }
  ]
}
```

## 모범 사례

- 주제 또는 기능별로 그룹을 나누어 논리적인 구조 유지
- 자주 사용하는 페이지는 앞쪽 그룹에 배치하고, 고급 콘텐츠는 접힘 기능 활용
- 그룹과 페이지에 적절한 아이콘 추가하여 시각적 경험 향상
- 페이지 수가 많은 그룹은 `collapsed: true` 설정 권장
