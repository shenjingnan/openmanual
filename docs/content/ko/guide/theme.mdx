# 테마 커스터마이징

OpenManual은 간결한 테마 커스터마이징 방안을 제공하며, `theme` 설정 필드를 통해 사이트 외관을 조정할 수 있습니다.

## 주 색상

`primaryHue` 색조 값을 통해 브랜드 색상을 조정합니다. 색조 값 범위는 0-360입니다:

```json
{
  "theme": {
    "primaryHue": 250
  }
}
```

### 자주 사용하는 색조 값 참조

| 색조 값 | 색상 | 효과 |
|--------|------|------|
| 0 | 빨간색 | 열정, 눈에 띔 |
| 30 | 주황색 | 활력, 따뜻함 |
| 60 | 노란색 | 밝음, 친근함 |
| 120 | 초록색 | 자연, 상쾌함 |
| 200 | 파란색 | 전문적, 신뢰감 |
| 250 | 보라색 | 창의적, 신비로움 |
| 300 | 분홍색 | 부드러움, 친밀함 |

## 다크 모드

기본적으로 다크 모드가 활성화되어 있으며, 설정을 통해 끌 수 있습니다:

```json
{
  "theme": {
    "darkMode": false
  }
}
```

## 전체 테마 설정

```json
{
  "theme": {
    "primaryHue": 250,
    "darkMode": true
  }
}
```

## 테마 설정 필드

| 필드 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `primaryHue` | number | — | 주 색상 색조 값 (0-360) |
| `darkMode` | boolean | `true` | 다크 모드 활성화 여부 |

## 정적 리소스

커스텀 정적 리소스(예: Logo, Favicon)를 `public` 디렉토리에 배치하세요:

```
my-docs/
├── openmanual.json
├── content/
└── public/
    ├── logo.svg
    └── favicon.ico
```

설정에서 참조:

```json
{
  "navbar": {
    "logo": "/logo.svg"
  }
}
```

## Logo 테마 적응

Light 및 Dark 모드에서 서로 다른 Logo를 사용해야 하는 경우, 객체 형식으로 설정할 수 있습니다:

```json
{
  "navbar": {
    "logo": {
      "light": "/logo-light.svg",
      "dark": "/logo-dark.svg"
    }
  }
}
```

`light`와 `dark` 경로가 다를 때, 네비게이션 바는 두 개의 `<img>` 태그를 렌더링하며 Tailwind의 `dark:hidden` / `hidden dark:block`을 이용해 자동으로 전환하여 표시합니다.

커스텀 Logo 이미지를 제공하지 않으면, OpenManual은 사이트 이름에 따라 자동으로 SVG Logo를 생성합니다. Dark 모드에서는 온난한 색조의 텍스트(`#E8E0D4`)를 사용하여 다크 테마 배경과 조화를 이룹니다.
