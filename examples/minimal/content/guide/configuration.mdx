# 配置

My Project 支持丰富的配置选项。

## 基本配置

| 选项 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| name | string | - | 项目名称 |
| port | number | 3000 | 服务端口 |
| debug | boolean | false | 调试模式 |

## 环境变量

- `MY_PROJECT_PORT` - 覆盖端口配置
- `MY_PROJECT_DEBUG` - 启用调试日志
